# HB 2562 - Local effort assistance

[Source](http://lawfilesext.leg.wa.gov/biennium/2025-26/Pdf/Bills/House%20Bills/2562.pdf)

## Section 1
(1) Beginning in calendar year 2020 2027 and each calendar year thereafter, the state must provide state local effort assistance funding to supplement school district enrichment levies as provided in this section.

## Section 2
This act takes effect January 1, 2027.
